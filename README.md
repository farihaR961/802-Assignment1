The program prompts the user to enter:
- Gmail address
- Gmail App Password
- Destination email address
- Full image path


File Included : email.m

Requirements

1. MATLAB installed 
2. Internet connection
3. Gmail account with:
   - 2-Step Verification enabled
   - Gmail App Password generated

How to Run

1. Open MATLAB.
2. Set the Current Folder to the directory containing: email.m
3. Run the file
4. Enter the requested information when prompted.

Example:

Gmail address: fariharahman961@gmail.com
Gmail App Password: xxxxxxxxxxxxxxxx
Destination email: fariha5@ualberta.ca
Full image path to attach: D:\802_A1\Question-3\test.jpg

Output

If successful, MATLAB displays:

"Email sent successfully to <destination_email>"

The destination email receives:
- Subject: part 4
- Message body
- Attached image file
